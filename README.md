# Usage
<!-- usage -->
```sh-session
$ npm install -g wutenv
$ wutenv COMMAND
running command...
$ wutenv (-v|--version|version)
wutenv/0.0.0 darwin-x64 node-v10.15.1
$ wutenv --help [COMMAND]
USAGE
  $ wutenv COMMAND
...
```
